# RAG_BASED_AI_ASSISTANT
RAG_BASED_AI_ASSISTANT is a voice-enabled RAG assistant that uses OpenAI's Whisper, GPT-4o-mini, and TTS to provide instant answers and exact YouTube timestamps from course videos. It features 2026 live web search and a 5-message memory to handle both course doubts and real-time news through a seamless voice-to-voice interface.
